Xcode利用git提交已创建项目

一般情况下,提交已有项目总是会出现项目名称后面的符号 ? ,正确的提交步骤如下:

  1. 在oschina或者github上创建新项目
  2. 创建完成后,命令行 git clone + 新项目地址 到本地文件夹,并且接着把已有的项目文件移动到该文件夹

  3. 利用终端进入到该工程的目录下

  4. 打开Xcode,点击 Source Control 点击 Push 即可提交所有代码
